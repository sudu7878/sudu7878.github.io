grapesOS

This project is almost like my dream come true. I always wanted to develop and ideate my own Operating System from scratch: which isn't based on any pre-existing projects such as the Linux kernel. grapesOS is made for ARM v64 chip, specifically for Raspberry Pi 4 due to its open source nature. I plan to make this project as a whole gaming ecosystem that has 3D-rendering libraries by-default, while keeping the efficiency and performance of ARM chips. Check it out on GitHub!
